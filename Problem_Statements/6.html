Matching Companies

In the previous sections, you did various types of analysis of company employee data. Most probably, you created a tree structure to represent the company hierarchy. (If not, you should have!)

Such a tree that captures the employee-manager relationship of a company is called an org-chart of the company (short for “organization chart”), and the employees that are directly under a manager are called the direct-reports of that manager. (i.e. A is a direct report of B if B is the manager of A.)

In this section, you have to describe an algorithm that you could use to find out if any two companies in the input file have the same org-chart structure. In other words, you have to find two companies in the input such that the two CEOs have the same number of direct-reports, and there is a mapping between the direct-reports of the first CEO to the direct-reports of the second CEO in such a way that the matching direct-reports have the same org-chart under them.

DO NOT write a program for this. Just describe how you will solve this problem. You may either give a text description, or you can write it as pseudo-code.

Bonus points if you can identify the technical name of a standard CS algorithm that can be used to solve this problem.

Warning: You will get only one attempt for this program, so make sure you submit a correct program. We recommend that you write the program in an IDE of your choice, compile and test it, and then submit. You will not get a chance to correct a bad program for this question.
