Number of Employees of Last Company

In the previous two sections, you did some analysis of the salaries of employees in some companies. In this section, you have to do a similar analysis - however, the input has now changed. The company names have been removed from the input, so input.txt now looks like this:

Vineel Phatak, 520, NOBODY
Ajay Joshi, 250, Vineel Phatak
Abhishek Chauhan, 120, Ajay Joshi
Jayesh Godse, 500, NOBODY
Vijaya Mundada, 60, Abhishek Chauhan
Shital Tuteja, 45, Jayesh Godse
Rajan Gawli, 700, Vineel Phatak
Zeba Khan, 300, Jayesh Godse
Chaitali Sood, 100, Zeba Khan
Sheila Rodrigues, 35, Vineel Phatak

As before, this input contains 2 companies - one with Vineel Pathak as the CEO and another with Jayesh Godse as the CEO. And, if you analyze the input carefully, you'll realize that the data is the same as before - there are 6 employees in Vineel's company (including Vineel), and there are 4 employees in Jayesh's company (including Jayesh). (Note: this is just an example input. The real input can contain any number of companies, and any number of employees.)

In this problem, you have to figure out the number of employees in the company of the last CEO mentioned in the input. Here, Vineel Phatak is the first CEO mentioned, and Jayesh Godse is the second (and last) CEO mentioned. And Jayesh's company has 4 employees (including Jayesh).

Thus, when your program is run, output.txt should contain:

4

All other rules remain the same as before.
