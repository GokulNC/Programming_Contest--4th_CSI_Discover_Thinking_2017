You are in Section 2: Simple Aggregate, question 1 of 1. There are a total of 6 sections.

Sum of Salaries

Consider a file input.txt which contains the employee hierarchy of various companies, in the following format:

Vineel Phatak, 520, Amkart, NOBODY
Ajay Joshi, 250, Amkart, Vineel Phatak
Abhishek Chauhan, 120, Amkart, Ajay Joshi
Jayesh Godse, 500, Airflix, NOBODY
Vijaya Mundada, 60, Amkart, Abhishek Chauhan
Shital Tuteja, 45, Airflix, Jayesh Godse
Rajan Gawli, 700, Amkart, Vineel Phatak
Sheila Rodrigues, 35, Amkart, Vineel Phatak
Zeba Khan, 300, Airflix, Jayesh Godse
Chaitali Sood, 100, Airflix, Zeba Khan

Each line contains an employee name, a salary (in Rs. thousands per month), the name of the company, and the name of the manager of that employee - all separated by commas. You may assume that employee name, company name, and manager name do not contain any commas, and that salary is an integer. The special manager name NOBODY indicates that this employee does not have a manager (and is hence the CEO of the company). You can assume that there is only one CEO in each company.

You have to write a program which reads input.txt (from the current directory), and finds the sum of all the salaries in the input. You should write that number to the file output.txt (in the current directory).

Thus, if input.txt contained the example input given above, your program should write the following to output.txt:

2630

Because that is the sum of all the 10 salaries in the input.

Please note the following:

  • Your program should read the input from a text file called input.txt in the current directory. Not C:\input.txt or some other name like that. If you can't figure out how to read input from input.txt in the current directory, you will not be able to solve this problem.
  • Your program must write output to a file called output.txt in the current directory. (Current directory is the directory in which your program is executed at runtime.) The output.txt must contain a single integer and nothing else.

Other things to note:

  • Please write your program in one of the supported languages (see the dropdown list at the bottom of this page).
  • Do not print anything extra to output.txt. Do not print prompts like "Please enter your input:"
  • If you write your program in C, please write in standard C. Do not use any Microsoft or Turbo-C specific functions (e.g. clrsrc, or getch), and do not include non-standard files (like conio.h).
  • If you write your program in Java, the main class in your program should be called public class Main. This class should contain the public static void main method. Please do not include a package declaration in your program.
  • If you write in C++, make sure to write standard C++. You may use the STL but please remember that you might need using namespace std; or equivalent in your code to make it work properly.

Note: Read the instructions carefully. Look at the example input and example output properly - that will ensure that you've understood the problem correctly. Most disqualified submissions are because of failure to follow the instructions exactly.

Note: if you're unable to solve this problem, you cannot proceed to the next section. Please log out, and you'll be evaluated on the basis of whatever you've submitted so far. “My program works on my computer, but is being rejected by the system” is not a valid complaint. That just means that you did not read the instructions carefully. Try again.
